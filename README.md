# Hotel Management Website 🏨

## 📌 Overview
A responsive hotel management website designed as a college project.  
Features include booking rooms, viewing hotel details, and user-friendly navigation.

## ⚙️ Tech Stack
- HTML, CSS, JavaScript
- SCSS for styling
